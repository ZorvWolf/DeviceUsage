# DeviceUsage :bar_chart:

DeviceUsage is an Android application that monitors:

- :brain: RAM usage (foreground + background apps)
- :signal_strength: Data usage (Daily / Monthly)
- :battery: Battery stats
- :gear: CPU usage

## :sparkles: Features
- Real-time RAM usage with circular graph
- Per-app RAM consumption with icons & names
- Daily / Monthly data usage
- Wi-Fi / Cellular toggle
- Modern Material UI

## :closed_lock_with_key: Permissions Required
- Usage Access
- Network Stats Access

The app automatically prompts users to enable required permissions.

## :tools: Built With
- Java
- Android SDK
- NetworkStatsManager
- ActivityManager
- Material Components

## :rocket: How to Run
1. Clone the repo
2. Open in Android Studio
3. Sync Gradle
4. Run on Android 7+ device

## :warning: Notes
- Some data usage stats require system permissions on certain OEM ROMs
- Values may vary depending on Android version
